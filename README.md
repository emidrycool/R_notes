# R_notes

## R语言介绍
R 语言是为数学研究工作者设计的一种数学编程语言，主要用于统计分析、绘图、数据挖掘。

## R环境安装
### base

用于基础分发的二进制文件。

### contrib

为过时版本的 R 提供的 CRAN 包的二进制文件（适用于 R < 4.0.x）。

### old contrib

提供的 CRAN 包的二进制文件（适用于 R >= 4.0.x）。

### Rtools

用于构建 R 和 R 包的工具。Rtools是用于编译和安装R包的工具。

### RStudio

RStudio 是一个 R 编程语言的集成开发环境（IDE）。它包括了一个控制台、语法高亮显示的编辑器，支持直接代码执行以及画图、浏览历史、调试和工作区管理的功能强大工具。

这个软件有两种格式：

- RStudio Desktop 作为常规的桌面应用
- RStudio Server 运行在远端服务器上，通过 Web 浏览器来使用。
